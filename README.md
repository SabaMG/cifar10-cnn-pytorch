### 1. Clone and create virtual environment

```bash
git clone https://github.com/SabaMG/cifar10-cnn-pytorch.git
cd cifar10-cnn-pytorch

```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Train the model

```bash

```

```bash
tensorboard --logdir=runs
```
